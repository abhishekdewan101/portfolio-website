Indoor location tracking Contextual infromation from bluetooth beacons

Project Description

The project was designed to create a platform that could derive and use contextual information from bluetooth beacons in oder to be able to improve the user experience of Samsung consumer products. The user's proximity to particular appliances would allow for tailoring the informations shown to the user on his/her phone or tablet.

Algorithm Overview

The result of the project was an algorithm that could accurately predict the location of a user within a given network of bluetooth beacons. The algorithm was based on the principles of mobile signal triangulation and past heuristics. The following two flowcharts accurately demonstrate the flow of the algorithm. The algorithm is current under review in the process of getting patented.

Mobile Application

In order to help track the user's location, a small android package was written that was capable of using the bluetooth hardware of the phone to ping the beacons and position itself within the given network of beacons. The application was a highly optimized piece of code running in the background of the user's phone and was in constant communication with the servers. The application's function was to keep a constant connection with all the different beacons present in the network and calculate the approximate distance of the user from these beacons. This data was constantly being uploaded to the servers.

Web Application

The web application served as a web interface to the data that was being dumped onto the servers. In order to demonstrate the working of the system, the web application was polling data from the servers and then trying to use the distance of the users and the pre-known location of the beacons to draw circles ( as illustrated in Image 1). The point of intersection of these circles provided the system with the best possible guess of the location of the user. To further improve accuracy the system measured the distance between the last known location of the user to the new locations to determine the shortest distance and hence provided the viewer with a more accurate position of the user.

Stalking Center

The stalking center was a interactive prototype written for Samsung TV's. Using the Samsung TV sdk, I was able to create web interface that could show the positions of the various beta users in the office space. As can be seen the interface shows you the last know location of the user which is updated when the user moves closer to another beacon denoting a different place.
