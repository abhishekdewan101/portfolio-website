AB Share Promxity based revocable file sharing platform.

Project Description

Using the indoor location tracking algorithm, this prototype was designed to create a proximity based file sharing platform for devices running the Android operating System. The idea was to use bluetooth beacons in order to determine the devices that are present in the proximity of a user and then allow seamless sharing of files between those devices. The sharing was drilled down to a single tap scenario, whereby one tap allowed the user to share his/her files. Moreover, the sharing platform also allowed users to revoke access to file that had already been shared.

Project Overview

Initial Design Concept

The concept of this project was to use the indoor locaton information obtained from the bluetooth beacons in order to provide the user with relevant contextual choices when he/she decides to share something from his/her device. The choices were made available based on the proximity of the other devices to the device of the user. The sharing action was also simplified to a simple "tap to share" scenario.

First Prototype

The video demonstrates the first working prototype of the platform. The middle phone is used to share a photo to two of the phones on either side. The photo is shared as long as the user does not revoke the sharing. As soon as the sharing is revoked the photo disappears from the other phones. At this point the sharing can only be done between Samsung smartphones.

Final Prototype

Expanding upon the first prototype, the reach of the devices was increased from phones to tablets cameras and TV's. Using the final prototype one could take a photo using a Samsung camera, running android, walk into a room with a Samsung phone and TV and wireless share the photo with said devices just in a matter of a single tap of the finger.The sharing can be removed and it wipes the shared photos off the other devices.
